# Pesquisa: APIs de Busca e Geração de Imagens para Squad de Conteúdo

**Data:** 2026-04-22
**Query:** APIs e sistemas de busca/geração de imagens para automação de conteúdo
**Contexto:** Squad de conteúdo precisa buscar imagens automaticamente para preencher templates de carrosséis e outros layouts

## TL;DR

Existem 3 categorias de soluções para o squad:

1. **Bancos gratuitos** (Pexels, Unsplash, Pixabay) — custo zero, SDKs oficiais em Node.js, qualidade curada
2. **Bancos pagos** (Shutterstock, Getty, Adobe Stock) — maior acervo, licenciamento comercial robusto, custo por imagem
3. **IA generativa** (DALL-E, Flux, Stable Diffusion) — imagens sob medida, $0.003-$0.19/imagem, sem dependência de acervo

**Recomendação:** Começar com Pexels + Unsplash (gratuitos, SDKs maduros), adicionar Flux Schnell ($0.003/img) para imagens personalizadas.
