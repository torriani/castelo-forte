# Query Original

## Pergunta

Quais APIs e sistemas de busca de imagens (gratuitos e pagos) podem ser integrados ao squad de conteúdo para preencher automaticamente templates de carrosséis e layouts com imagens relevantes?

## Contexto Inferido

- **Foco:** Técnico — integração de APIs de imagens em sistema de automação de conteúdo
- **Domínio:** Node.js/JavaScript (scripts de renderização de carrosséis existentes)
- **Temporal:** 2025-2026 (estado atual das APIs)
- **Caso de uso:** Squad de conteúdo gera templates HTML de carrosséis. Em alguns slides não há imagens, e o sistema precisa buscar/gerar imagens automaticamente baseado no tema do conteúdo.
