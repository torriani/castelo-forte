# Checklist de Elementos BELIEF

> Mínimo 2 de 5 elementos presentes por peça de conteúdo.

## ELEMENTOS BELIEF (marcar os presentes)

### 1. Confirmar Suspeitas
- [ ] Valida algo que a audiência já desconfiava
  - [ ] A suspeita mencionada é real e reconhecível (não inventada)
  - [ ] O leitor consegue se ver na situação descrita em até 5 segundos
  - [ ] Existe pelo menos 1 dado, exemplo ou história que comprova a suspeita
- [ ] Usa linguagem tipo "Você sempre soube que..." ou "Não é impressão sua..."
  - [ ] O tom é de validação, não de condescendência
  - [ ] A frase de abertura gera um "sim!" interno no leitor
- [ ] Traz dados ou exemplos que comprovam a suspeita
  - [ ] A evidência é específica (números, nomes, casos reais)
  - [ ] A fonte da evidência é credível ou verificável

### 2. Justificar Fracassos
- [ ] Explica por que a audiência não conseguiu antes
  - [ ] O motivo apresentado é lógico e faz sentido para o leitor
  - [ ] Não invalida os esforços anteriores da audiência
  - [ ] Conecta o fracasso a algo externo (método, informação, contexto)
- [ ] Tira o peso da culpa individual ("Não foi culpa sua")
  - [ ] O alívio é genuíno, não parece desculpa barata
  - [ ] A audiência se sente compreendida, não vitimizada
- [ ] Aponta o verdadeiro motivo do fracasso (sistema, método errado, desinformação)
  - [ ] O motivo apontado é verificável ou amplamente aceito
  - [ ] Existe uma transição clara entre "por que falhou" e "o que fazer agora"

### 3. Atirar Pedras (Inimigo Comum)
- [ ] Define um inimigo claro (pessoa, sistema, crença, indústria)
  - [ ] O inimigo é nomeado ou descrito com especificidade (não vago)
  - [ ] O inimigo não é o próprio leitor nem alguém que ele admira
  - [ ] O inimigo representa um obstáculo real ao resultado desejado
- [ ] Cria aliança "nós contra eles"
  - [ ] O leitor se sente parte de um grupo que "entende a verdade"
  - [ ] A aliança não depende de ódio, mas de posicionamento claro
- [ ] O inimigo é credível e identificável pela audiência
  - [ ] A audiência já teve experiência negativa com esse inimigo
  - [ ] A crítica ao inimigo é sustentada por fatos ou lógica

### 4. Aliviar Culpa
- [ ] Remove responsabilidade pessoal do fracasso
  - [ ] O leitor não se sente acusado em nenhum momento
  - [ ] A remoção de culpa é feita com empatia, não com pena
  - [ ] O alívio abre espaço para a nova solução (não fecha o argumento)
- [ ] Mostra que o problema é estrutural/sistêmico
  - [ ] O problema sistêmico é explicado de forma simples e clara
  - [ ] O leitor consegue ver como o sistema o prejudicou especificamente
- [ ] Usa tom de compreensão, não de julgamento
  - [ ] Relido em voz alta, o tom soa como aliado, não como professor
  - [ ] Não há superioridade implícita no texto

### 5. Encorajar Sonhos
- [ ] Pinta um futuro desejável e alcançável
  - [ ] O futuro descrito é específico (não genérico como "liberdade financeira")
  - [ ] O leitor consegue se imaginar nessa realidade
  - [ ] O prazo ou caminho para chegar lá é crível
- [ ] Mostra que a transformação é possível
  - [ ] Existe pelo menos 1 prova concreta (caso real, número, depoimento)
  - [ ] A transformação apresentada é proporcional ao esforço descrito
- [ ] Usa prova social ou exemplo concreto de resultado
  - [ ] A prova social é de alguém com perfil similar ao da audiência
  - [ ] O resultado apresentado inclui detalhes verificáveis (tempo, valores, contexto)

## VALIDAÇÃO FINAL
- [ ] Mínimo 2 dos 5 elementos estão presentes
- [ ] Elementos escolhidos são coerentes com o tema do conteúdo
- [ ] Nenhum elemento parece forçado ou artificial
- [ ] Tom emocional é autêntico, não manipulativo
