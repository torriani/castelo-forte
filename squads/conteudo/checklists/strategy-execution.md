# Checklist de Execução de Estratégia

## COLETA DE INFORMAÇÕES (4 obrigatórias)
- [ ] Mercado/nicho do cliente identificado
- [ ] Persona principal definida (dores, desejos, linguagem)
- [ ] Objetivo da campanha claro (awareness, engajamento, conversão)
- [ ] Restrições e preferências do cliente documentadas

## ESTRATÉGIA
- [ ] Estratégia correta identificada para o objetivo
- [ ] Justificativa da escolha documentada
- [ ] Métricas de sucesso definidas
- [ ] Riscos mapeados e mitigações planejadas

## CRONOGRAMA
- [ ] Cronograma completo com datas e horários
- [ ] Frequência de postagem definida
- [ ] Distribuição equilibrada entre formatos
- [ ] Datas de revisão/ajuste previstas

## TEMPLATES
- [ ] Templates prontos para cada dia do cronograma
- [ ] Cada template segue o padrão do formato (carrossel, reel, stories)
- [ ] Copys completas com hooks, corpo e CTA
- [ ] Briefings visuais incluídos quando necessário

## INTEGRAÇÃO FEED + STORIES
- [ ] Conteúdo de Feed e Stories se complementam
- [ ] Stories referenciam/amplificam posts do Feed
- [ ] Progressão narrativa coerente entre formatos
- [ ] Horários de publicação coordenados

## TOM E LINGUAGEM
- [ ] Tom imperial consistente em todas as peças
- [ ] Sem linguagem genérica ou clichê
- [ ] Vocabulário alinhado com a marca
- [ ] Autoridade presente sem arrogância

## CTAs
- [ ] Cada peça tem CTA claro
- [ ] CTAs contêm palavra-chave relevante
- [ ] Variação de CTAs entre peças (não repetitivo)
- [ ] CTAs direcionam para ação mensurável

## VALIDAÇÃO FINAL
- [ ] Validado pelo oráculo
- [ ] Feedback incorporado
- [ ] Versão final aprovada
- [ ] Pronto para execução
