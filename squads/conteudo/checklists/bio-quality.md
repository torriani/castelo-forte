# Checklist de Qualidade da Bio Imperial

## ESTRUTURA (3 fases obrigatórias)

### Fase 1: Inversão
- [ ] Subverte a expectativa do leitor
- [ ] Primeira frase causa estranhamento
- [ ] Quebra o padrão "ajudo X a fazer Y"

### Fase 2: Desafio
- [ ] Provoca o leitor a questionar suas crenças
- [ ] Cria tensão intelectual ou emocional
- [ ] Posiciona o autor como diferente do mercado

### Fase 3: Redefinição
- [ ] Redefine o que o autor faz de forma única
- [ ] Deixa claro o território que ocupa
- [ ] Fecha com autoridade, não com pedido

## REGRAS DE FORMATO
- [ ] Frase da Bio <= 150 caracteres
- [ ] Concisa e impactante (sem enrolação)
- [ ] Legível em tela mobile

## PALAVRAS PROIBIDAS
- [ ] Zero uso de "ajudo"
- [ ] Zero uso de "inspiro"
- [ ] Zero uso de "ensino"
- [ ] Nenhum verbo passivo/servil

## CRITÉRIOS DE QUALIDADE
- [ ] Não parece bio de LinkedIn
- [ ] Gera tensão ou curiosidade imediata
- [ ] Verbo forte utilizado (comandar, construir, destruir, revelar)
- [ ] Verdade contraintuitiva presente
- [ ] Diferencia claramente de qualquer concorrente
- [ ] Funciona sozinha (sem precisar ver o perfil)

## TESTE FINAL
- [ ] Alguém lendo pela primeira vez ficaria intrigado?
- [ ] Comunica autoridade sem listar credenciais?
- [ ] É memorável (fica na cabeça)?
