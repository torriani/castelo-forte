# Checklist de Qualidade de Hooks

## REGRAS OBRIGATÓRIAS
- [ ] Max 18 palavras (slide 1 / carrossel)
- [ ] Max 5 palavras (hook de Reel)
- [ ] JAMAIS começa com pergunta
- [ ] Contraintuitivo obrigatório
- [ ] Zero contexto — só tensão
- [ ] Primeira palavra inesperada (Reels)
- [ ] Nenhuma palavra proibida (segredo, dica, truque, hack, simples, fácil, rápido)
- [ ] Nenhuma frase proibida ("Vou ser honesto...", "A verdade é que...")

## PADRÃO VIRAL (pelo menos 1)
- [ ] Contraintuitivo
- [ ] Segredo/Revelação
- [ ] Confissão
- [ ] Urgência
- [ ] Prova Imediata
- [ ] Tribal (nós vs eles)
- [ ] Meta (quebra da 4a parede)

## VARIAÇÕES OBRIGATÓRIAS (carrossel)
- [ ] 3 variações geradas
- [ ] 2 variações VIRAL (do hooks-bank)
- [ ] 1 variação IMPERIAL (criada no momento)
- [ ] Formato S1_HOOKS seguido

## TESTE FINAL
- [ ] Gera reação: "Como assim?", "Isso é possível?", "Nunca ouvi falar"
- [ ] Pararia SEU scroll?
- [ ] Causa desconforto ou curiosidade?
