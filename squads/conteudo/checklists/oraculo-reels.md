# Checklist: Oráculo Reels

> Validação em 3 níveis. Score >= 80% em CADA nível.
> Progressão obrigatória: Nível 1 -> 2 -> 3.

---

## Nível 1: Copy

- [ ] Benefício forte e irresistível
- [ ] Track record / prova presente
- [ ] Credibilidade crível
- [ ] Ângulo diferenciador (ideia única)
- [ ] Útil, Urgente, Único, Ultraespecífico
- [ ] Especificidade brutal (números exatos, ímpares, contexto)
- [ ] 12 Testes do Validator aprovados
- [ ] Sem palavras proibidas (compliance 100%)
- [ ] Score Nível 1 >= 80%

## Nível 2: Hook

- [ ] Usa 1+ dos 7 padrões virais
- [ ] Max 5 palavras (>5 = reprovação automática)
- [ ] Primeira palavra inesperada (não "Você", "Eu", "A", "O")
- [ ] Texto na tela + falado
- [ ] Gera sentimento forte (Choque/Curiosidade/Raiva/Medo)
- [ ] Não revela solução (incompletude forçada)
- [ ] Score Nível 2 >= 80%

## Nível 3: Reel Completo

### Bloco 2 (Retenção)
- [ ] Gap de informação mantido
- [ ] Tensão narrativa com escalada
- [ ] Sem vale de energia
- [ ] Contexto relevante e conciso

### Bloco 3 (Segundo Hook)
- [ ] Re-engajamento presente (30-40s)
- [ ] Segmenta para ICP
- [ ] Novo conflito ou promessa renovada

### Bloco 4 (Conteúdo)
- [ ] Entrega valor real
- [ ] Elemento do Movimento presente (OBRIGATÓRIO)
- [ ] Metáfora/exemplo concreto
- [ ] Não técnico demais

### Bloco 5 (Moral + CTA)
- [ ] Gera 1 dos 5 sentimentos climáticos (OBRIGATÓRIO)
- [ ] Insight transformador
- [ ] CTA com comando claro
- [ ] UM único CTA
- [ ] Não vende diretamente
- [ ] Score Nível 3 >= 80%

## Resultado

- [ ] Score Geral >= 80%: APROVADO
- [ ] Score < 80%: REESCREVER
