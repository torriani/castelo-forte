# Checklist de Qualidade da Análise de Concorrentes

## COLETA DE DADOS
- [ ] Mínimo 5 concorrentes identificados por mercado/nicho
- [ ] Perfis dos concorrentes documentados (nome, seguidores, nicho exato)
- [ ] Top 10 posts coletados por concorrente (maior engajamento)
- [ ] Período de análise definido (últimos 30/60/90 dias)

## MÉTRICAS DOCUMENTADAS
- [ ] Curtidas médias por post registradas
- [ ] Comentários médios por post registrados
- [ ] Compartilhamentos (quando visível) registrados
- [ ] Taxa de engajamento calculada por concorrente
- [ ] Frequência de postagem mapeada
- [ ] Formatos mais usados identificados (carrossel, reel, imagem, stories)

## ANÁLISE DE PADRÕES
- [ ] Padrões de hooks identificados com exemplos concretos
- [ ] Temas recorrentes de conteúdo mapeados
- [ ] Estruturas de copy mais usadas documentadas
- [ ] Estilos visuais predominantes catalogados
- [ ] Horários de publicação mais comuns identificados
- [ ] Tom de voz de cada concorrente caracterizado

## GAPS E OPORTUNIDADES
- [ ] Gaps de conteúdo identificados (o que ninguém faz)
- [ ] Oportunidades de diferenciação listadas
- [ ] Temas sub-explorados pelo mercado mapeados
- [ ] Formatos pouco usados pelos concorrentes identificados
- [ ] Ângulos únicos possíveis documentados

## SUGESTÕES DE CONTEÚDO
- [ ] Mínimo 10 sugestões de conteúdo geradas
- [ ] Cada sugestão tem justificativa baseada na análise
- [ ] Sugestões adaptadas ao estilo Castelo Forte (tom imperial)
- [ ] Sugestões cobrem diferentes formatos
- [ ] Sugestões exploram os gaps identificados
- [ ] Prioridade/ordem de execução sugerida

## VALIDAÇÃO FINAL
- [ ] Dados são verificáveis (links/prints dos posts analisados)
- [ ] Análise é objetiva, não opinativa
- [ ] Insights são acionáveis (dão direção clara)
- [ ] Documento está organizado e fácil de consultar
