# Checklist de Coerência de Campanha Multi-Formato

## MENSAGEM CENTRAL
- [ ] Mensagem-chave única e clara definida
- [ ] Mensagem consistente entre todos os formatos (Feed, Stories, Reels)
- [ ] Nenhum formato contradiz ou enfraquece a mensagem central
- [ ] Adaptação de linguagem por formato sem perder a essência

## PROGRESSÃO EMOCIONAL
- [ ] Jornada emocional planejada entre as peças
- [ ] Progressão coerente (tensão -> desenvolvimento -> resolução)
- [ ] Cada peça contribui para o arco narrativo geral
- [ ] Não há saltos emocionais bruscos ou desconexos

## TOM IMPERIAL
- [ ] Tom imperial mantido no Feed
- [ ] Tom imperial mantido nos Stories
- [ ] Tom imperial mantido nos Reels
- [ ] Tom imperial mantido nos CTAs
- [ ] Sem quebra de registro (formal/informal inconsistente)
- [ ] Autoridade presente em todas as peças

## CTAs ALINHADOS
- [ ] CTAs direcionam para o mesmo objetivo final
- [ ] CTAs se complementam entre formatos
- [ ] Progressão lógica dos CTAs (awareness -> interesse -> ação)
- [ ] Nenhum CTA contradiz outro
- [ ] Palavra-chave consistente nos CTAs

## CRONOGRAMA INTEGRADO
- [ ] Feed e Stories publicados em horários complementares
- [ ] Sequência de publicação segue a progressão narrativa
- [ ] Espaçamento adequado entre peças
- [ ] Stories amplificam/referenciam posts do Feed
- [ ] Nenhuma sobreposição que canibalize atenção

## CONSISTÊNCIA VISUAL (quando aplicável)
- [ ] Paleta de cores coerente entre formatos
- [ ] Estilo visual reconhecível como mesma campanha
- [ ] Elementos visuais se complementam

## TESTE FINAL
- [ ] Se alguém visse apenas 1 peça, entenderia a mensagem?
- [ ] Se alguém visse todas as peças, a experiência é fluida?
- [ ] Nenhuma peça parece "solta" ou desconectada?
- [ ] A campanha como um todo é maior que a soma das partes?
