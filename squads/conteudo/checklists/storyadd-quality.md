# Checklist de Qualidade de StoryAds

## FORMATO DE TEXTO
- [ ] Máximo 3 linhas de texto
- [ ] Texto curto e direto
- [ ] Fonte e tamanho legíveis em mobile
- [ ] Sem blocos de texto densos

## ESTRUTURA NARRATIVA
- [ ] Elemento ANTES presente (situação inicial/dor)
- [ ] Elemento DEPOIS presente (transformação/resultado)
- [ ] Elemento INSINUAÇÃO presente (curiosidade sobre o como)
- [ ] Progressão Antes -> Depois -> Insinuação clara

## APARÊNCIA VISUAL
- [ ] Imagem parece casual/acidental
- [ ] Não parece produção profissional forçada
- [ ] Parece conteúdo orgânico de Stories
- [ ] Sem elementos gráficos excessivos (setas, emojis, badges)

## INTENCIONALIDADE
- [ ] Não parece intencional/planejado
- [ ] Passa como conteúdo espontâneo
- [ ] Sem linguagem de vendas explícita
- [ ] Sem urgência artificial ("ÚLTIMAS VAGAS", "SÓ HOJE")

## OBJETIVO
- [ ] Não vende diretamente
- [ ] Gera desejo genuíno
- [ ] Cria curiosidade sobre o próximo passo
- [ ] Deixa o espectador querendo mais informação

## TESTE FINAL
- [ ] Você passaria reto se visse nos Stories? (deve responder NÃO)
- [ ] Parece story de um amigo compartilhando algo real?
- [ ] Gera vontade de responder ou clicar?
- [ ] Funciona sem som (para Stories sem áudio)?
