# Checklist: Oráculo Posts

> Validação obrigatória antes de entregar qualquer carrossel.
> Score >= 80% para aprovar.

---

## Slide 1 (Hook)

- [ ] Max 18 palavras
- [ ] Sem pergunta (interrogação)
- [ ] Contraintuitivo
- [ ] Zero contexto, só tensão
- [ ] 3 variações geradas (2 virais + 1 imperial)
- [ ] Formato S1_HOOKS correto

## Slides 2-10

- [ ] Max 60 palavras por slide
- [ ] Cada slide adiciona informação nova
- [ ] Progressão emocional: Reptiliano (1-3) -> Límbico (4-6) -> Neocórtex (7-10)
- [ ] Tensão constante do início ao fim

## Linguagem

- [ ] Sem palavras proibidas (cliches-proibidos.md)
- [ ] Sem frases de rejeição imediata
- [ ] Sem clichês de coach
- [ ] Foco no leitor ("você"), nunca "eu/nós"
- [ ] Tom NÚCLEO Castelo Forte mantido

## Estrutura

- [ ] Tipo de post correto (estrutura slide-a-slide)
- [ ] Framework de copy aplicado
- [ ] Solução não revelada cedo demais
- [ ] Pelo menos 1 dos 5 efeitos obrigatórios presente
- [ ] Arco dramático com clímax e decisão

## CTA

- [ ] 3 opções geradas
- [ ] Sem "clique aqui", "me chama", "link na bio" (genérico)
- [ ] Comando moral + palavra-chave + promessa
- [ ] Alinhado com intenção do post

## 12 Testes do Validator

- [ ] 1. Autenticidade Emocional
- [ ] 2. Ausência de Marcadores
- [ ] 3. Presença Cinematográfica
- [ ] 4. Linguagem Cirúrgica
- [ ] 5. Profundidade Experiencial
- [ ] 6. Comando Militar
- [ ] 7. Tensão Narrativa
- [ ] 8. Valor x Brutalidade (70/30)
- [ ] 9. Estrutura Integrada
- [ ] 10. Variabilidade Expressiva
- [ ] 11. Coerência com Frameworks
- [ ] 12. Proteção Contra Vazamentos

## Resultado

- [ ] Score >= 80%: APROVADO
- [ ] Score < 80%: REESCREVER
