## Estrutura de Registro

Quando o performance-tracker registrar padroes, cada entrada seguira este formato:

### Padrao Confirmado
```
### [Nome do Padrao]
- **Posts analisados:** N
- **Engagement medio:** X%
- **Formato:** Carrossel / Reels / Single
- **Tipo de post:** Imperial / Polemico / Crenca / etc
- **Horario:** Faixa de horario
- **Tamanho:** N slides (se carrossel)
- **Insight:** Descricao do padrao
- **Acao recomendada:** O que fazer com essa informacao
```

### Winner
```
### [Data] — [Titulo/Hook do post]
- **Formato:** X
- **Engagement:** X% (Xpx acima da media)
- **Melhor metrica:** Saves / Shares / Comments
- **Por que funcionou:** Analise
```

### Loser
```
### [Data] — [Titulo/Hook do post]
- **Formato:** X
- **Engagement:** X% (Xpx abaixo da media)
- **Pior metrica:** X
- **Por que falhou:** Analise
- **Licao:** O que evitar
```
